## How to Play
Complete both levels to escape the wrath of the evil rat king! Avoid the rabid rats, coming in contact with one will result in death!
| Key          | Action          |
|:------------:|:---------------:|
| W            | Jump            |
| A            | Move left       |
| D            | Move right      |
| SPACE        | Shoot a grenade |
